# Optimizacion-Algoritmos-Recursivos
Proyecto para el análisis, predicción y optimización del rendimiento de algoritmos recursivos. Utiliza ecuaciones de recurrencia, el Teorema Maestro y pruebas experimentales en Java para comparar la complejidad teórica con el tiempo de ejecución, el consumo de memoria y la profundidad de la pila en algoritmos como MergeSort y QuickSort.
